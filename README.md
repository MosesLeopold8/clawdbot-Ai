# clawdbot-Ai
clawdbot是什么 Clawdbot（现更名为Moltbot）是一个开源的本地AI智能体项目，由Peter Steinberger发起。它旨在将AI能力与本地设备深度结合，实现更高效、个性化的任务执行和交互体验。以下是其核心特点和功能：  本地运行与隐私控制 Clawdbot运行在用户自己的设备（如Mac、Linux服务器或树莓派）上，数据和配置存储在本地，用户对数据拥有完全控制权，避免了云端服务可能带来的隐私风险。  多渠道交互 通过WhatsApp、Telegram、iMessage、Slack等聊天软件作为交互入口，用户可在熟悉的聊天界面中与AI对话，无需切换应用。
